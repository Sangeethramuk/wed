# EducAItors

An AI-powered educator dashboard built with Next.js 16, React 19, and Tailwind CSS.

## Structure

```
.
├── dashboard/      # Next.js app (deployed to Vercel)
└── References/     # Design reference HTML files
```

## Development

```bash
cd dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

This project is configured for [Vercel](https://vercel.com). The `vercel.json` at the root sets `dashboard/` as the root directory.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/N-Admn/educAItors)

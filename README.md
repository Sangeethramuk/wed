# EducAItors

An AI-powered educator dashboard built with Next.js 16, React 19, and Tailwind CSS.

## Structure

```
.
├── src/            # Next.js app source
├── public/         # Static assets
└── References/     # Design reference files
```

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
